## 0.1.0

- Complete FLAC metadata parsing from bytes and streams.
- Immutable document model with edit-through-return pattern.
- Vorbis comment CRUD operations (set, add, remove, clear).
- Picture block support (add, replace, remove, export).
- Padding inspection and management.
- Streaming transforms for memory-efficient large file processing.
- File I/O with atomic writes (temp file + rename).
- CLI with modern subcommands and `metaflac`-compatible flag aliases.
- JSON output, dry-run, batch processing, and modification time preservation.
- Complete DartDoc documentation for all public APIs.
- Runnable code examples for common workflows.
- Performance benchmarks for parsing, transforms, and memory usage.
- Migration guide from reference `metaflac` tool.
